# Nudge-ADHD-Device
---
##What is it
Nudge is a haptic feedback device made to combat ADHD-related time blindness. It does this by stimulating the user ever 15 minutes with a haptic effect.
---

##Why I made this project
I made this project because I, as an indivdual with ADHD who is heavily affected by time blindness, have missed a lot of opportunities, deadlines, and dates. (just now I missed the Stasis deadline to recieve a 150-point discount)
I hope that with this device, I, and others with ADHD, can create an artificial perception of time and live life with less regrets. 
---

##Future Plans
In the future I want to expand this device to not only be a Time-blindness solution, but an entire ADHD multitool. 
This means adding additional features such as calendar reminders, built-in focus timers, date countdowns, and a user-interactive display.
